## 🚀 Funcionalidades Principais

- **Busca Global**: Integração direta com a API do TMDB para encontrar qualquer mídia.
- **Armazenamento Offline**: Uso de IndexedDB para garantir que sua lista esteja disponível mesmo sem internet.
- **Gestão de Itens**: Adicionar, editar, remover e marcar conteúdos como assistidos.
- **Interface Moderna**: UI limpa com modais para detalhes, configurações e injeção de dados.
- **Segurança**: Inclui funcionalidade de `LockScreen` para privacidade da lista.

## 🛠️ Como Iniciar

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure sua chave da API do TMDB nas variáveis de ambiente.
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
